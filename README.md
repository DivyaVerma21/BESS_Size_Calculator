BESS Size Calculator

Overview
The BESS Size Calculator is a Streamlit web application that helps users optimize their Battery Energy Storage System (BESS) for peak shaving and price arbitrage. The tool integrates real-time Nordic electricity spot prices and allows users to input their consumption data manually or via CSV upload. The app calculates potential savings and visualizes energy consumption, grid load, and pricing trends.

Features
  Fetch Real-Time Spot Prices: Retrieves NO1 electricity spot prices from [hvakosterstrommen.no](https://www.hvakosterstrommen.no/).
  User Consumption Profile: Users can manually input hourly consumption data or upload a CSV file.
  Battery Parameters Input: Users specify battery power, capacity, efficiency, and state of charge (SoC) limits.
  Peak Shaving Optimization: Reduces grid import spikes to lower energy costs.
  Price Arbitrage Optimization: Determines optimal charge/discharge cycles based on spot prices.
  Savings Calculation: Estimates potential cost savings from peak shaving and arbitrage strategies.
  Data Visualization: Plots consumption patterns, net grid load, and spot prices.
